# XOProto（xochat Protocol）

## 1. 概述

xochat_im Protocol（XOProto）是一套基于TCP的二进制协议，用于客户端与服务端之间的通信。XOProto的设计目标是简单、高效、易扩展。

## 2. 使用

```
go get github.com/xochat/xochat_im_go_proto
```

```
import  (
    xoproto "github.com/xochat/xochat_im_go_proto"
    ...
)
```
